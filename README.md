# Codix (EdTech)
Codix es una propuesta de aplicación educativa diseñada para enseñar programación de forma interactiva, visual y progresiva.

## Objetivo del proyecto
El proyecto busca facilitar el aprendizaje de conceptos de programación mediante una experiencia amigable, combinando diseño, contenido educativo y una interfaz moderna.

## Características principales
- Enfoque visual y didáctico- Diseño orientado a estudiantes- Estructura web moderna- Integración de recursos multimedia- Cuestionarios interactivos

## Tecnologías utilizadas
HTML  CSS  JavaScript
